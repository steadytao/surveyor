# Support Policy

Surveyor is open-source software maintained on a best-effort basis.

The latest released version is the only supported release line unless a narrower support statement is published for a specific release.

## Where to Ask for Help

Use GitHub Discussions for:
- questions about usage
- broader feedback on the repo, docs or command surface
- design ideas and feature suggestions
- discussion around use cases and workflow fit

Use GitHub Issues for:
- concrete bugs
- release blockers
- tracked implementation work

Use private reporting for:
- security vulnerabilities in Surveyor itself, as described in [`SECURITY.md`](SECURITY.md)

## Response Expectations

Surveyor is a personal project. Response times are best effort and there is no guaranteed support SLA.

## Scope of Support

Support covers Surveyor itself. It does not include general consulting on scanned environments, PKI migrations or unrelated third-party systems.
