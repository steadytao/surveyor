Surveyor v0.1.0 captures the initial TLS inventory MVP work. It turns the repository from a set of internal packages into a usable command-line tool for explicit TLS targets, with conservative reporting around observed cryptographic posture.

## Highlights

- Added an initial `surveyor scan tls` CLI that can scan targets from a YAML config file or from explicit `host:port` values passed on the command line.
- Added TLS inventory collection, X.509 metadata extraction, conservative readiness classification and both JSON and Markdown reporting.
- Added deterministic fixtures, golden output tests and cross-platform CI so the current MVP is verifiable and stable rather than just functional.

## Security

No known security fixes are included in this release.

## In this release

- Surveyor now supports explicit TLS-facing targets only. It does not yet attempt discovery across ranges or cover non-TLS cryptographic surfaces.
- Output is available as canonical JSON and derived Markdown.
- Release automation now builds downloadable binaries for Linux, macOS and Windows on amd64 and arm64.
- The release workflow closes the matching milestone automatically once the release is published.
